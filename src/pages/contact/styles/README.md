# ContactPage Styles

This directory splits `src/pages/ContactPage.css` into concern-focused files.

## File map

- `00-tokens.css`: page tokens, shared headings, and global contact-page color setup
- `01-hero.css`: intro section, primary desk card, and highlight cards
- `02-layout.css`: office rail, sticky form, and field styling
- `03-responsive.css`: responsive stacking and reduced-motion adjustments

## Editing guideline

Keep shared tokens early and layout-specific overrides later so the sticky form and office rail remain predictable.
